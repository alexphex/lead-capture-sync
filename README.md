# Lead Capture Sync

A WordPress plugin that synchronizes leads with external automation platforms such as n8n, HubSpot, and Google Sheets.

## Features

- Contact Form 7 integration
- REST API
- Secure Webhooks
- Retry mechanism
- Logging
- CRM synchronization

## Requirements

- PHP 8.2+
- WordPress 6.x
- Composer

## Installation

```bash
composer install
```
